## 2) 视觉风格锚点词典（6 种 style key）

### `live-action`

- 英文注入词：
  `photorealistic, cinematic film quality, real human actors, professional cinematography, natural lighting, 8K resolution, shallow depth of field, film grain texture, color graded, anamorphic lens flare, three-point lighting setup`
- 中文简描：
  `真人实拍电影风格，photorealistic，8K高清，专业摄影`

### `anime`

- 英文注入词：
  `Japanese anime style, cel-shaded, vibrant saturated colors, large expressive eyes with detailed iris highlights, dynamic action poses, clean sharp outlines, consistent line weight throughout, Studio Ghibli/Makoto Shinkai quality, painted sky backgrounds, soft ambient lighting with dramatic rim light`
- 中文简描：
  `日本动漫风格，cel-shaded，鲜艳色彩，Studio Ghibli品质`

### `2d-animation`

- 英文注入词：
  `classic 2D animation, hand-drawn style, Disney/Pixar quality, smooth clean lines with consistent weight, expressive characters with squash-and-stretch principles, painterly watercolor backgrounds, soft gradient shading, warm color palette, round friendly character proportions`
- 中文简描：
  `经典2D动画风格，手绘风格，Disney/Pixar品质`

### `3d-animation`

- 英文注入词：
  `high-quality 3D CGI animation, Pixar/DreamWorks style, subsurface scattering on skin, detailed PBR textures, stylized character proportions, volumetric lighting, ambient occlusion, soft shadows, physically-based rendering, motion blur`
- 中文简描：
  `3D CGI动画，Pixar/DreamWorks风格，精细材质`

### `cyberpunk`

- 英文注入词：
  `cyberpunk aesthetic, neon-lit urban environment, rain-soaked reflective streets, holographic UI displays, high-tech low-life contrast, Blade Runner style, volumetric fog with neon color bleeding, chromatic aberration, cool blue-purple palette with hot pink and cyan accents, gritty detailed textures`
- 中文简描：
  `赛博朋克美学，霓虹灯光，未来科技感`

### `oil-painting`

- 英文注入词：
  `oil painting style, visible impasto brushstrokes, rich layered textures, classical art composition with golden ratio, museum quality fine art, warm undertones, Rembrandt lighting, chiaroscuro contrast, canvas texture visible, glazing technique color depth`

## 4) 三视图（正/侧/背）生成提示词

用途：为角色生成一致性的三视图设定图（常用于角色设计定稿、建模前参考）。

来源改写自：
- `services/ai/visualService.ts`（角色九宫格 turnaround 提示词）
- `docs/BigBanana-AI-Director-提示词全集.md`（B-7 角色九宫格造型图片）

### 英文通用模板（推荐）

```text
Create ONE character turnaround sheet in a 1x3 layout (3 equal panels with thin white separators, no labels or panel numbers).
All panels must show the SAME character; only view angle changes.

Visual Style: {visualStyle} ({stylePrompt})
Character: {characterName} - {characterSummary}

Panels (left to right):
Panel 1: [Front View / Full Body] - {frontDescription}
Panel 2: [Side View / Full Body] - {sideDescription}
Panel 3: [Back View / Full Body] - {backDescription}

Constraints:
- Output ONE single 1x3 image only (do NOT generate multiple images)
- Face consistency: facial proportions, eye shape, nose, mouth, ears must be identical across all 3 panels
- Hair consistency: hairline, length, volume, parting style must match exactly
- Body consistency: height, build, limb proportions must be preserved
- Outfit consistency: clothing design, colors, textures, accessories (belt, shoes, jewelry, bags) must be exactly the same
- Lighting: consistent light source direction and intensity across all panels
- Color palette: identical color grading and tone in all 3 panels
- Background: clean neutral solid color (white, light gray, or mid-gray), no shadows on background
- Composition: full-body shot, character centered, same scale in all panels
- Each panel should be clear, reference-quality, and well composed with good proportions
{artDirectionSuffix}

Top priority: The character must look unmistakably like the SAME person in all 3 panels. Any inconsistency in face, hair, outfit, or proportions is unacceptable.
```
