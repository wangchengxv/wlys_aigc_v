# AIGC 实训平台执行清单

## 当前节奏

- 范围基线：以 `aigc-server + aigc-site-react` 作为正式主线工程。
- 交付原则：优先补齐平台基础与教学闭环，不推翻已有创作主链路。
- 当前状态：平台基础第一批能力、“课程 / 作业 / 提交 / 评分”最小教学闭环、第一版“正式账号登录体系 + RBAC 权限接入”、“组织 / 班级 / 用户归属管理”、“对象存储抽象与统一媒体资源管理”、以及生产闭环第一批能力（`TTS / 配音`、`口型同步`、`成片编排`、`导出包交付`、`内容审核`）与第一版 `统计看板` 已落地，当前重点转向运营治理深化，包括成本分析、模型消耗统计与失败告警。

## 仓库核对结果

- 已完成：平台基础前 6 项、正式账号登录体系、RBAC 角色模型、组织 / 班级 / 用户归属管理、对象存储抽象与统一媒体资源管理、教学闭环最小链路、课程模板教学接入、审计日志查询页、`TTS / 配音工作流`、`口型同步工作流`、`成片编排与镜头拼接`、`导出包 / 成片交付`、`内容审核流与驳回机制`、`统计看板` 均已在代码中落地。
- 已验证：后端新增账号密码登录、JWT、角色与资源归属权限；教师可查看学生提交并评分，但不可删除学生作品；新增组织 / 班级 / 用户目录管理接口与管理页；新增统一媒体存储抽象、`LOCAL` provider 与媒体资源目录页；新增项目级配音、口型同步、成片编排、导出包、审核流和统计看板页面与接口；后端测试与前端构建已通过。
- 未完成：`成本分析与模型消耗统计`、`失败补偿与告警`、`评论协作`、`版本对比与回滚增强` 等运营治理能力仍未落地。

## 最近完成

- 账号与权限：已完成账号密码登录、JWT、`ADMIN / TEACHER / STUDENT` 角色接入，以及教学链路与剧本工程的资源归属权限判断。
- 教学权限：教师现在可查看相关学生提交并进行评分 / 点评，但不能删除学生作品；学生仍只保有本人作品的可写权限。
- 组织治理：已补齐组织、班级、用户目录的后端数据模型、管理接口与前端管理页，管理员可维护组织 / 班级 / 用户归属。
- 媒体资源：已补齐统一媒体存储抽象、`stored_file_record` 存储 provider / bucket / object key 元数据，以及管理员可见的媒体资源目录页。
- 生产闭环：已补齐项目级配音、口型同步、成片编排、导出包交付与内容审核流，形成“生成 -> 交付 -> 审核”的最小生产闭环。
- 运营治理：已新增第一版统计看板，可查看课程、作业、提交、项目、导出包、待审核项目总览、状态分布与最近活动摘要。
- 验证结果：后端集成测试已覆盖登录、教学权限、目录管理、媒体资源、配音、口型同步、成片编排、导出包、审核流与统计看板；前端构建通过，可进入登录页、设置页、目录管理页、媒体资源页、生产闭环页面和统计看板联调。
