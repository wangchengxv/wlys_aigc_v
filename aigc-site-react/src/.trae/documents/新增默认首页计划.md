# 实现计划：新增默认首页（未登录用户欢迎界面）

## 需求概述
为未登录用户创建一个精美的默认首页/登录页面，UI 风格与现有平台保持一致（紫色+青色渐变主题），模仿传统登录界面但更加美观现代化。

## 设计方向
- **美学风格**：现代简约 + 科技感，使用渐变、光效和微动效
- **主色调**：保持现有平台的紫色 (#5e40df) + 青色 (#06b6d4) 渐变
- **字体**：Plus Jakarta Sans（现有字体）
- **特色**：
  - 全屏渐变背景 + 光效装饰
  - 居中的登录卡片面板
  - 快速账号填充按钮（admin/teacher/student）
  - 第三方登录入口（OneLinkAI）
  - 平台功能展示预览

## 实现步骤

### 步骤 1: 创建 `WelcomePage.tsx`
创建新的欢迎页组件，包含：
- 全屏渐变背景 + 装饰性光效元素
- 居中的欢迎/登录面板
- 登录表单（用户名、密码）
- 快速登录账号预设按钮
- 第三方登录入口
- 平台功能简览（可选）

### 步骤 2: 创建 `WelcomePage.css`
创建对应的样式文件，包含：
- 全屏布局样式
- 背景渐变和装饰光效
- 登录面板样式
- 表单样式
- 动画效果

### 步骤 3: 更新 `router.tsx`
修改路由配置：
- 当用户未登录时，根路径 `/` 渲染 `WelcomePage`
- 当用户已登录时，根路径 `/` 渲染 `HomePage`
- `/login` 路由保持重定向到 `/?login=1`

### 步骤 4: 更新 `AppLayout.tsx`（如需要）
调整布局逻辑以支持新的首页渲染方式。

## 文件清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `src/pages/WelcomePage.tsx` | 新增 | 欢迎页主组件 |
| `src/styles/welcome-page.css` | 新增 | 欢迎页样式 |
| `src/router.tsx` | 修改 | 添加条件路由逻辑 |
| `src/layouts/AppLayout.tsx` | 修改（如需要） | 布局调整 |

## 技术要点
- 使用 `useAuthStore` 获取当前登录状态
- 根据 `user` 是否存在决定渲染哪个页面
- 复用现有的 `LoginModal` 样式变量（如 `--primary`, `--secondary`）
- 复用 `AppButton`, `AppInput` 组件保持一致性
