# AI视频生成风格与提示词大全

AI 视频生成的效果很大程度上取决于提示词的精准描述，不同的视觉风格需要搭配对应的关键词与场景描述，才能生成符合预期的视频内容。本文整理了当前主流的 AI 视频风格，以及可直接套用的完整提示词模板，帮你快速上手 AI 视频创作。

## 通用提示词公式

无论使用哪个 AI 视频平台（通义万相、Runway、Pika、可灵等），都可以套用这个通用公式来构建提示词，确保要素完整：

```Plain Text

[主体] + [动作] + [环境] + [运镜] + [光影] + [风格] + [画质/禁忌]
```

- 主体：视频的核心对象，越具体越好

- 动作：主体的动态行为

- 环境：场景、时间、背景

- 运镜：镜头的运动方式（推进、环绕、跟拍等）

- 光影：光线、色调、氛围

- 风格：视觉风格标签

- 画质 / 禁忌：清晰度、稳定性，以及避免的问题（无穿模、无畸形等）

## 提示词写作注意事项

1. **避免过于笼统**：不要只写 "美女视频"，要细化主体、动作、场景的细节

2. **避免元素堆砌**：每个提示词控制 3-4 个核心要素，不要同时加入太多冲突的风格

3. **要素逻辑匹配**：风格与色调、灯光要匹配，比如赛博朋克不要搭配复古暖色调

4. **加入禁忌词**：一定要加上 "无穿模、无畸形、无模糊、画面稳定" 这类词，减少 AI 生成的错误

5. **适配平台**：国内平台（通义万相、可灵）优先用中文提示词，海外平台（Runway、Pika）可以用英文关键词辅助
