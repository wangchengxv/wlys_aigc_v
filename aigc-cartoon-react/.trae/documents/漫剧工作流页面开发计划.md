# 漫剧工作流页面开发计划

## 任务目标
根据 Figma 设计稿生成 React 单文件组件，100% 还原"漫剧工作流"页面。

## Figma 设计稿信息

- 节点 ID: `175:75`

## 实现步骤

### Step 1: 创建页面组件文件
- 文件路径: `src/pages/WorkflowPage.tsx`
- 使用 React + CSS 文件方式
- 遵循项目现有组件风格

### Step 2: 创建样式文件
- 文件路径: `src/styles/workflow-page.css`
- 使用绝对定位 + Flexbox 混合布局
- 响应式适配 (1920px 设计稿为基础)

### Step 3: 实现核心 UI 结构
1. 背景层 (全屏图片)
2. 顶部导航 (Logo + 头像)
3. 左侧步骤导航 (垂直排列的步骤项)
4. 底部操作栏 (胶囊按钮组)

### Step 4: 添加交互逻辑
- 步骤选择状态管理
- 按钮点击事件占位
- 路由跳转占位 (如需要)

### Step 5: 样式精细化
- 字体加载 (Lobster 字体)
- 透明度背景处理
- 圆角和间距精确还原

## 技术选型
- React 18 + TypeScript
- CSS Modules 风格 (`.css` 文件)
- 项目已有依赖: antd, @ant-design/icons

## 交付物
1. `src/pages/WorkflowPage.tsx` - React 组件
2. `src/styles/workflow-page.css` - 样式文件
