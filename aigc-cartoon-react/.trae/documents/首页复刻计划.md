# 首页界面复刻计划

## 目标
将 `pages/FigmaMainPage.tsx` 的界面内容移植到首页路由，点击导航栏"首页"后显示该界面，整体主题改为黑色（暗色主题）。

## 具体步骤

### 1. 创建新的 HomePage 页面组件
- 路径：`src/pages/HomePage.tsx`
- 从 `pages/FigmaMainPage.tsx` 移植所有功能和UI逻辑
- 修改为黑色主题样式
  - 背景改为深色（#0a0a0a 或 #111）
  - 文字改为浅色
  - 卡片/按钮等改为暗色调
  - 渐变背景改为深色系

### 2. 创建 HomePage 样式文件
- 路径：`src/styles/home-page.css`
- 基于 `figma-main-page.css` 修改为暗色主题
  - `.figma-page` 背景改为深色背景
  - 输入框、按钮等组件改为暗色系
  - 文字颜色调整

### 3. 修改路由配置
- 文件：`src/App.tsx`
- 添加首页路由：`path="/" -> HomePage` 或保持 `index` 指向 HomePage
- 将首页导航的 key 指向正确的路径

### 4. 更新导航栏配置
- 文件：`src/layouts/BasicLayout.tsx`
- 确保"首页"导航项正确映射到 HomePage

### 5. 更新 BasicLayout 样式
- 确保暗色主题整体协调
- 导航高亮样式与黑色主题匹配

### 6. 构建验证
- 运行 `npm run build` 确保无编译错误
