# 创作页面工作流步骤 UI 优化计划

## 目标
优化创作页面顶部五个工作流按钮（全局设定、剧本、主体、分镜、剪辑成片）的布局和视觉设计。

## 当前实现分析
- 当前使用简单的水平按钮排列
- 样式基础，缺乏视觉层次和吸引力
- "剧本"按钮被标记为 active 状态

## 设计方向
采用 **科技感/影视化风格**，契合AI视频创作平台的主题：
- 深色背景配合渐变光效
- 步骤之间有视觉连接线/进度指示
- 悬停和激活状态有精致动效
- 使用圆角胶囊按钮配合图标和微光效果

## 实现步骤

### 1. 更新 TSX 结构
- 为每个步骤添加序号显示
- 添加步骤之间的连接线元素
- 增强按钮的视觉层级（图标 + 文字 + 序号）
- 添加步骤进度指示器

### 2. 更新 CSS 样式
- 设计胶囊式按钮外观，带渐变边框和发光效果
- 添加步骤间连接线（虚线或渐变条）
- 实现悬停时的光晕动画效果
- 添加激活状态的呼吸灯效果
- 使用伪元素创建装饰性背景光效

### 3. 动效设计
- 悬停时光晕扩散效果
- 激活按钮的脉冲呼吸动画
- 步骤间的连接线渐变动画
