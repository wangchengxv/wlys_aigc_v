# 全站删除页面底部说明区块计划

## Summary
- 目标：删除所有页面底部的“查看…说明/提示”区块（`HelpHint` 折叠说明），包括你提到的“查看课程中心说明”“查看项目库说明”等。
- 范围：仅删除底部说明 UI，不改业务流程、权限逻辑、接口请求和页面主功能。

## Current State Analysis
- 当前“底部说明”统一组件：
  - `aigc-site-react/src/components/common/HelpHint.tsx`（`details/summary` 结构）。
- 当前存在两种注入方式：
  - 页面直接渲染 `<HelpHint ...>`：
    - `aigc-site-react/src/pages/MediaResourcesPage.tsx`
    - `aigc-site-react/src/pages/HomePage.tsx`
    - `aigc-site-react/src/pages/TeachingCourseDetailPage.tsx`
    - `aigc-site-react/src/pages/TeachingCoursesPage.tsx`
    - `aigc-site-react/src/pages/OperationsDashboardPage.tsx`
    - `aigc-site-react/src/pages/ScriptProjectListPage.tsx`
    - `aigc-site-react/src/pages/SettingsPage.tsx`
  - 项目子页壳层统一渲染 `help`：
    - `aigc-site-react/src/components/script/ProjectSubpageShell.tsx`（`{help ? <HelpHint .../> : null}`）
    - 该壳层覆盖 `ScriptProjectPreview/Assets/Video/Dubbing/LipSync/FinalComposition/Export/PromptTemplates` 等页面的底部说明。

## Proposed Changes

### 1) 移除项目子页壳层底部说明渲染
- 文件：`aigc-site-react/src/components/script/ProjectSubpageShell.tsx`
- 变更内容：
  - 删除 `HelpHint` 引入与底部渲染片段。
  - 保留 `helpTitle/help` props（可选）或视实际清理程度决定是否同步删掉类型字段；以“最小改动优先”。
- 原因：
  - 一次性清除所有项目子页底部说明，避免逐页改动。

### 2) 删除页面直连的 HelpHint 区块
- 文件：
  - `aigc-site-react/src/pages/MediaResourcesPage.tsx`
  - `aigc-site-react/src/pages/HomePage.tsx`
  - `aigc-site-react/src/pages/TeachingCourseDetailPage.tsx`
  - `aigc-site-react/src/pages/TeachingCoursesPage.tsx`
  - `aigc-site-react/src/pages/OperationsDashboardPage.tsx`
  - `aigc-site-react/src/pages/ScriptProjectListPage.tsx`
  - `aigc-site-react/src/pages/SettingsPage.tsx`
- 变更内容：
  - 删除 `<HelpHint ...>...</HelpHint>` 区块。
  - 清理对应的 `HelpHint` import，避免 lint 报未使用。
- 原因：
  - 满足“所有页面底部说明都删除”。

### 3) 保留 HelpHint 组件文件（暂不删）
- 文件：`aigc-site-react/src/components/common/HelpHint.tsx`
- 变更内容：
  - 组件文件不删除，仅确保不再被页面使用。
- 原因：
  - 最小风险策略，避免误伤潜在未扫描到的动态/延迟使用点；后续可按需再做 dead code 清理。

## Assumptions & Decisions
- 决策 1：你要删除的是“页面底部说明折叠区块”，不是删除页面内全部说明文案（例如表单字段说明、状态提示仍保留）。
- 决策 2：工作台页此前已移除“查看工作区提示”，本次不重复处理。
- 决策 3：本次按“全站当前可见页面”执行；若后续新增页面，遵循同规则不再添加底部 `HelpHint`。
