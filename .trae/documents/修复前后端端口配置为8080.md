# 修复前后端端口配置为 8080

## 问题描述
Java 后端当前端口为 8081，需要改回 8080；前端也已配置连接 8080 端口。

## 当前状态
| 组件 | 当前端口 | 配置文件 |
|------|----------|----------|
| Java 后端 (`aigc-server`) | **8081** | `aigc-server/src/main/resources/application.yml:2` |
| Go 后端 (`aigc-server-go`) | 8080 (默认) | `aigc-server-go/internal/config/config.go:127` |
| 前端 (`aigc-site-react`) | 连接 8080 | `aigc-site-react/.env:2` |

## 修复步骤

### Step 1: 修改 Java 后端端口配置
- **文件**: `aigc-server/src/main/resources/application.yml`
- **修改**: 将 `server.port: 8081` 改为 `server.port: 8080`

### Step 2: 确认前端端口配置（无需修改）
- **文件**: `aigc-site-react/.env`
- **当前配置**: `VITE_API_BASE_URL=http://localhost:8080` ✅ 已正确

### Step 3: 验证
- 重启 Java 后端服务
- 确认后端监听 8080 端口
- 前端应能正常连接后端
