# 左侧导航退出与顶部栏隐藏改造计划

## Summary
- 目标：按你要求完成 3 项 UI 调整：
  - 将顶部栏“退出”按钮迁移到左侧导航栏右下角；
  - 删除创作工作台中的“查看工作区提示”模块；
  - 隐藏整个顶部栏（不是只隐藏“工作区”文字）。
- 范围：仅前端 UI 层调整，不改接口、不改鉴权逻辑。

## Current State Analysis
- 顶部栏与“退出”按钮位置：
  - `aigc-site-react/src/layouts/AppLayout.tsx` 中 `header.app-shell__topbar` 包含左侧“工作区”文案和右侧账号/退出按钮。
  - 当前退出逻辑：点击后调用 `signOut().then(() => navigate('/login'))`。
- 左侧导航结构与底部区域：
  - `aigc-site-react/src/components/layout/AppShellNav.tsx` 的 `app-shell-nav__footer` 当前只显示用户信息（姓名与用户名/提示文案），未承载操作按钮。
  - 样式容器在 `aigc-site-react/src/styles/global.css`：`.app-shell-nav`、`.app-shell-nav__footer` 已是纵向布局，适合放置底部动作区。
- “查看工作区提示”位置：
  - `aigc-site-react/src/pages/WorkspacePage.tsx` 使用 `<HelpHint title="查看工作区提示">...</HelpHint>`。

## Proposed Changes

### 1) 退出按钮迁移到左侧导航右下角
- 文件：
  - `aigc-site-react/src/layouts/AppLayout.tsx`
  - `aigc-site-react/src/components/layout/AppShellNav.tsx`
  - `aigc-site-react/src/styles/global.css`
- 变更内容：
  - 在 `AppLayout` 中移除顶部栏内的“退出”按钮渲染。
  - 将退出动作（`signOut + navigate('/login')`）通过 props 传入 `AppShellNav`，由导航底部渲染一个退出按钮。
  - 在 `app-shell-nav__footer` 内新增“信息块 + 退出按钮”布局；按钮右对齐并贴近底部右侧（右下角视觉位置）。
- 原因：
  - 满足“退出功能放到左侧导航栏右下角”。
  - 保持退出行为与当前一致，不引入逻辑回归风险。

### 2) 隐藏整个顶部栏
- 文件：
  - `aigc-site-react/src/layouts/AppLayout.tsx`
  - `aigc-site-react/src/styles/global.css`
- 变更内容：
  - `AppLayout` 不再渲染 `header.app-shell__topbar`（整块移除）。
  - 调整 `app-shell__main` / `app-shell__content` 间距，避免顶部栏移除后留空或错位。
  - 保留页面主体 `PageToolbar + Outlet` 正常展示。
- 原因：
  - 你明确选择“隐藏整个顶部栏”。

### 3) 删除“查看工作区提示”模块
- 文件：
  - `aigc-site-react/src/pages/WorkspacePage.tsx`
- 变更内容：
  - 删除 `HelpHint` 引入与 `<HelpHint title="查看工作区提示">...</HelpHint>` 片段。
  - 保持其余工作台结构（统计条、入口切换、Dock 面板）不变。
- 原因：
  - 满足“删除查看工作区提示功能”。

## Assumptions & Decisions
- 决策 1：仅迁移顶部栏退出按钮，不改 `SettingsPage` 内“退出登录”条目位置。
- 决策 2：顶部栏整块隐藏后，不再提供顶部菜单按钮；桌面端依赖左侧导航固定展示。
- 决策 3：只删除工作台页该提示块，不影响其他页面 `HelpHint`（例如首页、课程页、设置页）。
- 决策 4：退出按钮文案保持“退出”，不改为“退出登录”。

## Verification steps
- 结构与交互验证：
  - 左侧导航底部右下角可见“退出”按钮，点击后成功登出并跳转 `/login`。
  - 顶部栏整体不再显示（无“工作区”、无顶部账号区、无顶部退出按钮）。
  - 工作台页不再出现“查看工作区提示”折叠块。
- 样式验证：
  - 左侧导航底部信息与退出按钮在不同屏宽下不重叠、不溢出。
  - 页面主内容顶部无异常空白。
- 工具验证：
  - `cd aigc-site-react && npm run lint`
  - `cd aigc-site-react && npm run build`
