# 左侧导航栏收起功能实现计划

## 需求概述
为 `ScriptProjectWorkflowNav` 左侧导航栏添加收起/展开按钮，关闭后右侧主功能区自动等比例放大至全屏模式。

## 实现方案

### 1. 修改 `ScriptProjectWorkflowNav.tsx`

**状态管理：**
- 添加 `collapsed` 状态（boolean，默认 false）
- 添加折叠时保存状态的逻辑（可选 localStorage 持久化）

**UI 改造：**
- 在导航栏头部 `wf-sidebar__head` 添加一个收起/展开切换按钮
- 折叠状态时，只显示一个紧凑的图标按钮（可点击展开）
- 展开状态时，显示完整的导航内容

**代码结构：**
```tsx
// 新增折叠状态
const [collapsed, setCollapsed] = useState(false)

// 切换函数
const toggleCollapsed = () => setCollapsed(prev => !prev)

// 添加按钮到 wf-sidebar__head
<button 
  className={`wf-sidebar__toggle${collapsed ? ' is-collapsed' : ''}`}
  onClick={toggleCollapsed}
  aria-label={collapsed ? '展开导航' : '收起导航'}
>
  <ChevronIcon />
</button>
```

### 2. 修改 `ProjectSubpageShell.tsx`

**布局改造：**
- 将 `collapsed` 状态提升到父组件或使用 Context
- 接收 `sidebarCollapsed` prop
- 根据折叠状态动态调整 `script-project-workflow-layout` 的 grid 布局

### 3. 修改 `components.css`

**新增样式：**

## 改动文件清单

1. `aigc-site-react/src/components/script/ScriptProjectWorkflowNav.tsx` - 添加折叠状态和切换按钮
2. `aigc-site-react/src/components/script/ProjectSubpageShell.tsx` - 传递折叠状态并调整布局
3. `aigc-site-react/src/styles/components.css` - 添加折叠相关的 CSS 样式

## 关键设计决策

1. **动画过渡**：使用 CSS transition 确保折叠/展开动画流畅
2. **图标旋转**：收起按钮内的箭头图标随状态旋转 180°
3. **最小化视图**：折叠后仅显示 logo 图标，点击可快速展开
4. **持久化**（可选）：折叠状态可保存到 localStorage，用户刷新页面后保持偏好
