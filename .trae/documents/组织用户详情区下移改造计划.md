# 组织用户详情区下移改造计划

## Summary
- 目标：将“组织与用户”页面中的详情区从右侧栏移动到页面下方，形成“上方筛选+表格+概览，下方详情”的单列阅读流。
- 范围：仅调整 `AdminDirectoryPage` 布局与相关样式，不改详情内容、筛选逻辑、表格交互、弹窗逻辑和接口调用。

## Current State Analysis
- 现状结构（`aigc-site-react/src/pages/AdminDirectoryPage.tsx`）：
  - 主体采用 `admin-directory-page__layout` 双栏结构；
  - 左栏为筛选区、表格区、组织概览；
  - 右栏 `aside.admin-directory-page__aside` 为详情区（含基础信息、校园身份、日志入口）。
- 现状样式（`aigc-site-react/src/styles/components.css`）：
  - `.admin-directory-page__layout` 使用 `grid-template-columns: minmax(0, 1.55fr) minmax(320px, 0.9fr)`；
  - `.admin-directory-page__aside` 为 `position: sticky; top: 84px;`，固定在右侧；
  - 详情区视觉样式本身已完整，可复用到下方位置。
- 问题：
  - 右侧详情导致横向分栏强，主操作区（表格）与详情区争夺视觉焦点；
  - 在中小屏和高密度内容下，右侧信息阅读连续性较差。

## Proposed Changes

### 1) 页面结构改为单列流
- 文件：`aigc-site-react/src/pages/AdminDirectoryPage.tsx`
- 改动：
  - 移除右侧 `aside.admin-directory-page__aside` 在双栏中的位置；
  - 将详情区模块整体移动到主内容区底部（组织概览之后）；
  - 保留现有详情按钮（查看日志、完整审计）、启停按钮、加入批量操作等逻辑不变。
- 预期：
  - 信息流自上而下：筛选 -> 表格 -> 概览 -> 详情；
  - 操作与阅读路径更一致，减少左右跳视。

### 2) 样式从“双栏+sticky”改为“单列+底部详情”
- 文件：`aigc-site-react/src/styles/components.css`
- 改动：
  - 将 `.admin-directory-page__layout` 改为单列（`grid-template-columns: minmax(0, 1fr)`）；
  - `.admin-directory-page__aside` 取消 `sticky` 语义，改为普通块级流；
  - 补充详情区与上方模块之间间距，避免视觉拥挤；
  - 保持 `.admin-directory-page__detail`、`.admin-directory-page__detail-hero`、`.admin-directory-page__detail-section` 现有视觉风格。
- 预期：
  - 无右侧悬浮栏，占宽更稳定；
  - 表格和详情都能使用全宽阅读。

### 3) 响应式断点一致性校正
- 文件：`aigc-site-react/src/styles/components.css`
- 改动：
  - 清理/同步现有 `@media` 下对 `admin-directory-page__layout`、`admin-directory-page__aside` 的双栏假设；
  - 确保移动端与平板端都保持“详情在下方”。
- 预期：
  - 各断点下布局一致，不出现回退到右侧栏的情况。

## Assumptions & Decisions
- 决策 1：只做位置迁移，不删减详情功能内容。
- 决策 2：详情区仍依赖当前选中行 `activeRow`，交互保持“点表格行更新详情”。
- 决策 3：日志仍通过弹窗查看，详情主卡不重新引入日志长列表。
- 决策 4：不改 API 与状态字段，避免引入数据层风险。

## Verification steps
- 功能验证：
  - 打开 `/admin/directory`，确认详情区在页面下方，不在右侧；
  - 点击表格行，底部详情内容正确更新；
  - “查看日志”“完整审计”“启用/停用当前账号”“加入批量操作”均可正常使用。
- 布局验证：
  - 页面在桌面/平板/移动端均为单列流，详情区始终在表格下方；
  - 不出现 sticky 侧栏残留或异常空白。
- 工具验证：
  - `cd aigc-site-react && npm run lint`
  - `cd aigc-site-react && npm run build`
