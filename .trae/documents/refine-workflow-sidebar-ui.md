# 生产流程侧边栏 UI 优化计划

## 问题分析
用户反馈当前侧边栏"还不完善，还有些卡顿"，需要：
1. **性能优化** - 解决卡顿问题
2. **视觉细节** - 完善 UI 设计
3. **动画流畅** - 优化过渡效果

## 实施步骤

### Step 1: 组件性能优化
- [ ] 使用 `useCallback` 优化 `getStepState` 函数
- [ ] 添加 `React.memo` 包装 `StepItem` 子组件
- [ ] 减少不必要的重渲染

### Step 2: 视觉细节完善
- [ ] 优化步骤圆点的发光效果和阴影层次
- [ ] 改进进度条的视觉设计（添加渐变粒子效果）
- [ ] 优化连接线的渐变和阴影
- [ ] 完善 Header 区域的设计（图标、背景纹理）
- [ ] 优化引导提示卡片的样式

### Step 3: 动画效果优化
- [ ] 优化进度条填充动画（使用更平滑的缓动）
- [ ] 添加步骤进入动画（staggered reveal）
- [ ] 优化 hover 过渡效果
- [ ] 添加当前步骤的呼吸动画

### Step 4: CSS 样式优化
- [ ] 合并重复样式规则
- [ ] 优化 z-index 层级
- [ ] 添加 GPU 加速属性（transform, will-change）
- [ ] 简化复杂的选择器

## 预期效果
- 消除卡顿，60fps 流畅渲染
- 视觉效果精致、有层次感
- 动画过渡自然流畅
- 与整体平台风格保持一致
