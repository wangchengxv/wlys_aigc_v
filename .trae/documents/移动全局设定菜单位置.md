# 移动"全局设定"到"项目生产"下的"项目库"下方

## 任务目标
将侧边栏中的"全局设定"从"工作台"分组移动到"项目生产"分组下的"项目库"菜单项下方。

## 涉及文件
- `/aigc-site-react/src/components/layout/AppShellNav.tsx`

## 修改步骤

### 1. 修改 AppShellNav.tsx

在 `SHELL_NAV` 配置中：
- **删除**：从 `workspace` 分组中移除 `{ label: '全局设定', to: '/global-settings' }`
- **添加**：在 `projects` 分组的 `items` 数组中，在 `项目库` 条目之后添加 `{ label: '全局设定', to: '/global-settings' }`

### 修改后的 projects 分组结构：
```typescript
{
  id: 'projects',
  title: '项目生产',
  items: [
    { label: '项目库', to: '/script-projects', patterns: ['/script-projects/*'] },
    { label: '全局设定', to: '/global-settings' },
    { label: '流程工作区', to: '/workflow/script-story', patterns: ['/workflow/*'] },
  ],
},
```

## 注意事项
- 路由 `/global-settings` 保持不变（位于顶层路由），无需修改 router.tsx
- 仅调整侧边栏导航的菜单层级结构
