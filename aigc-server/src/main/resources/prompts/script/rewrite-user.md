## 改写规则（严格遵守）
1. 核心保留：保留核心故事线、主题立意、主要人物设定，不颠覆主旨，不增删主要角色。
2. 结构优化：确保情节因果清晰，按铺垫、激励事件、冲突升级、高潮、余波增强结构。
3. 场景衔接：优化场景过渡，增强道具、台词、情绪或视觉元素的隐性关联。
4. 对话提升：对白自然立体、角色语言风格明确，并承载信息传递、人物塑造、情感表达、潜台词四重功能。
5. 冲突强化：强化利益、价值观、情感冲突，设置关键冲突爆发点与合理反转。
6. 人物深化：补足内心动机和心理弧光，通过动作、表情、停顿、沉默体现情绪层次。
7. 节奏调控：做到张弛有度，删减冗余，保证每幕每场有明确戏剧功能。
