# TESTS - 质量底线（自检表）

## 1. 测试分层
- 前端单元：Vitest（工具函数、Hooks）。
- 前端组件：React Testing Library（表单、卡片、弹窗、任务进度）。
- 前端 E2E：Playwright（核心业务链路）。
- 后端单元：JUnit5 + Mockito（Service 逻辑）。
- 后端接口：Spring Boot Test + MockMvc（鉴权、参数、返回）。
- 集成测试：Testcontainers（MySQL/Redis）+ 模型适配 Mock Server。

## 2. 发布前最小用例（必须通过）
- 登录成功后可创建项目。
- 未登录访问项目接口返回 401。
- 用户无法访问他人项目（越权拦截）。
- 项目名为空或重复时创建失败。
- 剧本生成成功返回 taskId，任务状态可轮询。
- 主体提取结果可落库并可编辑定稿。
- 分镜排序后 `shot_index` 连续且正确。
- 分镜图/视频成功后资产表有对应记录。
- 成片导出任务完成后可下载成片。
- 失败任务可查看原因并可重试。
