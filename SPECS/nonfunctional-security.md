# Nonfunctional & Security

## 1. 性能目标
- 首页首屏：首次 <= 3s，二次 <= 1.5s。
- 项目列表：分页查询响应 <= 2s（默认 20 条）。
- AI 任务创建：接口返回 taskId <= 1s。
- 任务进度：轮询间隔 2-5s，或采用 SSE 推送。

## 2. 安全要求
- 认证：Spring Security + JWT。
- 授权：项目、剧本、主体、分镜、资产都必须做 `user_id` 归属校验。
- 上传：严格校验类型、大小、扩展名。
- 密钥：模型 API Key 仅后端存储，不下发前端。
- 审计：删除项目/资产/成片、修改模型配置必须有操作日志。

## 3. 可扩展性要求
- 模型通过统一适配层接入，新增厂商不改核心业务代码。
- 视觉风格模板配置化管理。
- 任务类型可扩展到音乐、口型、动作捕捉等能力。
- 存储层兼容 MinIO/OSS/COS/S3。

## 4. 可用性要求
- AI 任务失败必须展示原因和重试入口。
- 用户离开页面后后台任务继续执行。
- 任务完成有通知机制。
- 表单未保存离开时必须有拦截确认。
- 批量任务支持取消。
