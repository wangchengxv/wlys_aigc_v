# Product Scope

## 1. 产品定位
Miioo 是面向动画短剧与 AI 视频创作团队的端到端创作平台，覆盖从创意输入到成片导出的完整流程。

## 2. 用户角色
- 普通创作者：快速产出短视频。
- 专业编剧：优化剧情结构与台词。
- 分镜师：细化镜头语言。
- 剪辑师：完成时间线与导出。
- 管理员：模型与系统配置管理。

## 3. 核心业务流程
```text
首页 -> 新建项目 -> 项目设定 -> 剧本 -> 主体 -> 分镜 -> 剪辑导出 -> 资产复用
```

## 4. 功能域范围
- 首页与导航
- 项目管理
- 剧本管理
- 主体管理
- 分镜管理
- 剪辑成片
- 资产库
- 模型配置
- 任务中心
- 用户与权限

## 5. 迭代目标
- 一期：登录、项目、剧本、主体、图片生成与资产入库。
- 二期：智能分镜、分镜图/视频、任务中心、资产库增强。
- 三期：时间线剪辑、字幕音频、成片导出、多模型与协作能力。
